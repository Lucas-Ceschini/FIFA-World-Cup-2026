# FIFA World Cup 2026 - Monte Carlo Simulation Metadata
# Generated: 2026-05-09
# Description: Documentation for all JSON files and scripts in the World Cup simulation project

## OVERVIEW
This project simulates the FIFA World Cup 2026 using Monte Carlo methods across multiple phases:
- Group Stage: 1000 simulations of group rankings
- Elimination Phase: 100 simulations of knockout tournament
- Champion Probabilities: Market-based odds for tournament winner

## SCRIPTS

### models/groups.py
Simulates group stage outcomes using Monte Carlo methods.
- Input: data/bet365_group.json (scraped betting odds)
- Output: Multiple JSON files with simulation results
- Method: Estimates team strength from odds, adds random noise, ranks teams

### models/elimination.py
Simulates knockout tournament phase.
- Input: data/bet365_champions.json (champion odds), data/classified.json (qualified teams)
- Output: Tournament simulation results
- Method: Uses market probabilities to simulate match outcomes

### models/combinations.py
Generates possible slot combinations for knockout tournament.
- Output: data/third_combinations.json
- Method: Creates all valid mappings of group winners to tournament slots

## INTERPRETATION NOTES

### Performance Scores
- Higher scores = better simulated performance
- Scores are relative within each simulation
- Not comparable across different simulations

### Probability Fields
- All probability fields are fractions (0.0 to 1.0)
- Calculated as: count / N_SIMULATIONS
- "eliminated" includes all teams that did not qualify

### Count Fields
- Raw scenario counts for verification
- Should sum appropriately across event types

### Best Third Rule
- Top 8 third-place teams qualify for knockout
- Selection based on simulated score among thirds

### Slot Combinations
- Maps group winners to tournament positions
- Ensures valid tournament bracket structure
- Third-place teams assigned to specific slots

## TECHNICAL DETAILS

### Group Stage Simulation
1. Estimate team strength from market probabilities
2. Normalize strengths by group
3. Add random noise using t-distribution
4. Rank teams within each group
5. Apply best third rule

### Elimination Phase Simulation
1. Use champion market odds as strength measure
2. Adjust for tournament phase difficulty
3. Simulate matches using probabilistic model
4. Track tournament progression

### Data Sources
- Bet365 betting odds (scraped HTML)
- Official FIFA group compositions
- Normalized market probabilities

### Reproducibility
- Fixed random seeds ensure identical results
- Same input data produces same output
- Change SEED for different sequences
